How to Install Windows 11 on Unsupported PC

Posted on March 29, 2022 by Geeky Coder | Updated: March 29, 2022
Category: Tutorials

Reading Time: 5 minutes

Are you stranded from getting the new awesome looking UI of Windows 11? Is your PC specs not up to the required specifications to run Windows 11? Well you've clicked on the perfect link to provide solution for you.

Download Windows

You can download Windows from the official Microsoft website here.

After donwloading windows using Media Creation tool or downloading the .iso file directly from the website, you will after to download .bat file which runs the command prompt in order to disable the sytem specifications checker (Note: This tutorial works for only the TPM 2.0, secure boot and processor bypass, if your PC is lacking in other specifications then this tutorial won't work for you. If your PC is lacking in other specifications different from those mentioned above, you can check on what to do here to Install Windows 11), now let's get started.
